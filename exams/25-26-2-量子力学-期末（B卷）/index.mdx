## 三、简答题：（1-2小题每题6分，3小题8分，共20分）

### 1、

假设高能态为 $\Phi_m$ 态，对应的粒子数为 $N_m$；低能态为 $\Phi_k$ 态，对应的粒子数为 $N_k$，写出应用受激发射现象实现的两个器件名称，简述什么是粒子数反转。

### 2、

设某原子体系分立能谱的能级由小到大排列：$\varepsilon_1 < \varepsilon_2 < \cdots < \varepsilon_k < \cdots < \varepsilon_m < \cdots$，为了描述原子在 $\varepsilon_m$ 和 $\varepsilon_k$ 两能级间的跃迁概率，爱因斯坦引入了三个系数：自发发射系数 $A_{mk}$、受激发射系数 $B_{mk}$、受激吸收系数 $B_{km}$，简述这三个系数及它们的意义。（设作用于原子的光波在 $\omega \to \omega + \mathrm{d}\omega$ 频率范围内的能量密度是 $I(\omega)\mathrm{d}\omega$）

### 3、

简述自然界的两类微观粒子，它们的自旋、体系波函数的特点以及它们所服从的统计分布。

## 四、证明题：（1-3小题每题5分，共15分）

### 1、

已知 $[\hat{\sigma}_x, \hat{\sigma}_y] = 2i\hat{\sigma}_z$，$\hat{\sigma}_x\hat{\sigma}_y + \hat{\sigma}_y\hat{\sigma}_x = 0$，证明：$\hat{\sigma}_x = \hat{\sigma}_y = \hat{\sigma}_z = i$。

### 2、

在自旋态 $\chi_{\frac{1}{2}}(S_z)$ 中，利用测不准关系证明 $\overline{(\Delta S_x)^2(\Delta S_y)^2} \ge \frac{\hbar^4}{16}$。

### 3、
