## 四、计算题

<Figure src="第四题图.png" float>
    第四题图
</Figure>

（5分）如图所示电路，当 $R_{L}=3\Omega$ 时获得最大功率3W，求 $R$ 的大小和此时电流源供出的功率P。

<Solution>
    答：负载 $R_L$ 左边等效电阻为 $2 + R$。（1 分）

    当 $2 + R = R_L = 3$ 时，有最大功率传输，故 $R = 1~\Omega$。（1 分）

    $U_{\text{oc}} = R i_s$

    $\dfrac{U_{\text{oc}}^2}{4 R_s} = \dfrac{(R\cdot i_s)^2}{4 R_L} = 3\ \text{W}$

    所以$i_s$=6A(1 分),此时电流源右边纯电阻电路等效电阻为 $(2+3)\parallel 1 + 2 = \dfrac{17}{6}\ \Omega$，（1 分）

    故功率 $P = i_s^2 \cdot \dfrac{17}{6} = 6^2 \cdot \dfrac{17}{6} = 102\ \text{W}$，（1 分）
</Solution>

（12分）求如图所示ab两端的二端网络的戴维南等效电路和诺顿等效电路，给出计算过程并画出戴维南等效电路图和诺顿等效电路图。

<Solution>
    求开路电压（共 4 分，可以看步骤给分）

    开路时， $10 i_1 * 3 - 12 + 6 i_1 = 0$， $i_1 = 1/3\ \ A$

    $U_{OC}$ 为 3 欧电阻上的电压 $U_{OC} = U_{ab} = 10 i_1 * 3 = 10\ V$

## 六、计算题

<Figure src="第六题图.svg" float>
    第六题图
</Figure>

（15分）在如图所示的电路中，$t < 0$时电路处于稳态，$t = 0$时开关由1位切换到2位，求$t≥0$时的与电容连接的等效电阻，时间常数$τ$，$u_{C}(t)$的零输入响应，零状态响应和全响应。

<Solution>
    解：$u_C(0^+) = u_C(0^-) = -2/(1+2+2)*5 = -2\,V$ （3 分）

    $u_C(\infty) = 2/(2+2)*10 = 5\,V$ （3 分）

    等效电阻 $R = 2k\,\Omega \parallel 2k\,\Omega = 1k\,\Omega$ （2 分）

    时间常数 $\tau = RC = 4*10^{-3}\,s$ （2 分）

    零输入响应 $u_C(t)_{zir} = u_C(0^+) (e^{-\frac{t}{\tau}}) = -2(e^{-250t})\,V$ （2 分）

    零状态响应 $u_C(t)_{zsr} = u_C(\infty)\,(1 - e^{-\frac{t}{\tau}}) = 5\,(1 - e^{-250t})\,V$ （2 分）

    全响应 $u_C(t) = u_C(t)_{zir} + u_C(t)_{zsr} = (5 - 7e^{-250t})\,V$ （1 分）
</Solution>

## 九、计算题

<Figure src="第九题图.svg" float>
    第九题图
</Figure>

（15分）如图所示二端网络N，已知$u(t)=40\cos (100t+60°)V$，$i(t)=10\cos (100t+90°)A$。请写出网络N的瞬时功率P(t)的表达式，求网络N吸收的平均功率，无功功率Q，视在功率S和功率因素$λ$。

<Solution>
    解：$P(t) = 40\cos(100t + 60^\circ) * 10\cos(100t + 90^\circ)$  （3 分）

    $P = UI\cos\varphi_{UI} = \frac{40}{\sqrt{2}} \frac{10}{\sqrt{2}} \cos(30^\circ) = 100\sqrt{3}\ \text{W}$  （3 分）

    $Q = UI\sin\varphi_{UI} = \frac{40}{\sqrt{2}} \frac{10}{\sqrt{2}} \sin(-30^\circ) = -100\ \text{VAR}$  （3 分）

    $S = UI = \frac{40}{\sqrt{2}} \frac{10}{\sqrt{2}} = 200\ \text{VA}$  （3 分）

    $\lambda = \cos\varphi_{UI} = \cos(30^\circ) = \sqrt{3}/2$  （3 分）
</Solution>
